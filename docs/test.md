# 二、关于 Docusaurus 的基本知识

- Docusaurus 是使用 React 来开发的。我们需要以 Markdown 的格式来撰写文档 & 博客，docusaurus会帮助生成静态网站。这其中涉及几个专业名词，如React、Markdown、静态网站。React我不懂，Markdown将写一篇笔记进行介绍。
- 使用 Algolia 工具，可以实现网站的全局搜索
